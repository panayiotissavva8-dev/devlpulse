// what is devpulse

Your GitHub activity,
finally worth showing off

Most developer portfolios are static. DevPulse is alive — it listens to your GitHub pushes via webhook, updates your stats in real time, and broadcasts changes to anyone viewing your profile over WebSocket.

One profile URL. One badge for your README. Zero maintenance.

See a live example →

// how it works

Up and running in 3 steps

No config files. No CLI tools. Just GitHub.

01 /

Sign in with GitHub

OAuth in one click. We pull your public repos, language data, and recent activity immediately after login.

02 /

Add the webhook

Copy your unique webhook URL from the dashboard and paste it into any repo's webhook settings on GitHub.

03 /

Push & watch it update

Every commit fires the webhook. Your profile updates live — stats, activity feed, streak counter, everything.
